##  Getting Started

### Django Backend

```bash
cd backend
python -m venv env
source env/bin/activate  # or `env\Scripts\activate` on Windows
pip install -r requirements.txt
python manage.py runserver
Flutter Frontend
```

### Flutter Frontend
```bash
cd frontend
flutter pub get
flutter run
```
